# 快速上手

本节将介绍如何在项目中使用 @oasis-end/utils。

## 版本

@oasis-end/utils 目前还处于快速开发迭代中。

[Shiki](https://github.com/joo1es/utils)

**Input**

````md
```js{4}

  data () {
    return {
      msg: 'Highlighted!'
    }
  }

```
````
